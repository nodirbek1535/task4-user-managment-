# Frontend (React + Vite)

## English

This is the client application for the User Management System.

### Main responsibilities
- Register and login flows
- Email confirmation page handling
- Protected route rendering
- User table and bulk user actions (block/unblock/delete)
- Authentication token handling with Axios interceptors

### Commands
```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

### API connection
Default API base URL is defined in:
- `src/api/axiosClient.js`

Update this value for local or production environments.
